## Що роблять всі файли

#### Ingestion.ipynb pipeline rag

- Підтягує документи з папки downloaded_docs/
- розділяє на чанки
- зберігає метадані для чанків
- побудувати векторний індекс в ChromaDB

#### Retrieval.ipynb

- Використовує similarity пошук
- і використовує покращену стратегію пошуку гібридний пошук

#### Answering.ipynb

- Створення детального запиту до rag моделі, щоб відповідала тільки на основі знайденого контексту, вивід знадених джерел (провірка на правдивість), і створення промпту як діяти моделі при різних ситуаціях

#### Evaluation.ipynb and Evaluation_question.json:

- Створення тестових питань для провірки
- Тестова провірка моделі, і вивід метрик

#### UI and FAstAPI

- FastAPI щоб запустити модель на сервер
- UI для інтерфейсу
